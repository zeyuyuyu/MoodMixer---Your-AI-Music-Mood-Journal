# MoodMixer

MoodMixer is an engaging web app that lets users track their daily mood through music. Users select their current emotion and receive an AI-curated playlist matching their mood. The app creates beautiful visualizations of their 'mood melody' over time and lets them share their emotional journey through music.

Key Features:
- Interactive mood selection through an intuitive emoji wheel
- Real-time music recommendations based on mood
- Beautiful animated visualizations
- Social sharing of mood-music combinations
- Weekly mood tracking insights

Tech Stack:
- HTML5/CSS3 with modern animations
- Vanilla JavaScript for core interactivity
- Canvas API for visualizations
- Local Storage for persistence
